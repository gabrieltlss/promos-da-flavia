# Promos da Flávia

[*Promos da Flávia*](https://promos.flaviatlss.com.br) é uma página para exposição de produtos com links para lojas afiliadas.
A página principal introduz o usuário, permite a visualização de produtos do dia e das categorias de produtos.
A página de categorias organiza os produtos por categorias distintas e permite a visualização de todo o catálogo de produtos.


## Tecnologias e link

EJS, CSS, W3CSS (framework), JavaScript, Mysql2 e express.
Hospedagem: ReselHost
